Aplicação do 2˚ Encontro GTS-CE
=

Este projeto está ligado a apresentação Agile Testing, entregando valor do início ao fim e foi criada durante a própria [apresentação](http://www.slideshare.net/elias.nogueira/agile-testing-entregando-valor-do-incio-a-fim).

Se você não está familiarizado com git, veja [este vídeo](http://youtu.be/jcyT_Cyak2Y) (5min) que vai te ensinar a baixar este projeto (37MB) e importa-lo ao [Eclipse IDE](http://www.eclipse.org).

O GTS-CE, Grupo de Teste de Software do Ceará pode ser conhecido através:
 * Lista de e-mails: https://groups.google.com/d/forum/teste-software-ce
 * Site: http://www.gtsce.com.br
